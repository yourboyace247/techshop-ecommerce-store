# TechShop E-commerce website.

### Building an online e-shop using HTML, CSS & JavaScript.

# :house: Home Page!
![Home Page](https://raw.githubusercontent.com/yourboyace247/techshop-ecommerce-store/main/WebsiteUI/HP.png)

Home page of the website. Banner adverting the shop, and a slide show showcase a few of the products.

# :iphone: Products!
![Products](https://raw.githubusercontent.com/yourboyace247/techshop-ecommerce-store/main/WebsiteUI/Products.png)

Products page. Items load from the products.json file.

### Category:

Option Category: You can filter the products by category (Laptops, Smartphones, Tables, & Accessories).

### Sort by:

Option Sort by: You can sort the products by price (Low to High, High to Low), & Name A-Z.

### Add to Cart:

Button Add to Cart: You can add a product to your cart.

# :dollar: Cart!
![Cart](https://raw.githubusercontent.com/yourboyace247/techshop-ecommerce-store/main/WebsiteUI/Cart.png)

Shopping Cart. Displays the items you added to your cart.<br>
Button "-" reduces the amount of the item, if you only have one item and you click it, the item gets deleted.<br>
Button "+" adds the amount of the item.<br>
Button "Delete" completely delets the item from your cart.<br>
Order summary shows you pricing details about the purchase.<br>
"Proceed to Checkout" button gets you to the next page "Shipping".

# :mailbox_with_mail: Shipping Page!
![Shipping](https://raw.githubusercontent.com/yourboyace247/techshop-ecommerce-store/main/WebsiteUI/Shipping.png)

Shipping page.You need to submit your shipping details to proceed with "Continue to Payment" button.<br>
"Go Back To Cart" gets you to the previous page shopping cart.<br>
You also have the order summary details on this page as well.

# :credit_card: Payment!
![Payment](https://raw.githubusercontent.com/yourboyace247/techshop-ecommerce-store/main/WebsiteUI/Payment.png)

Payment page.It requires credit or debit card details in order for the "Place Order" to work.<br>
"Back to Shipping" button gets you to Shipping Page.

# :white_check_mark: Thank you!
![ThankYou](https://raw.githubusercontent.com/yourboyace247/techshop-ecommerce-store/main/WebsiteUI/ThankYou.png)

Thank You for Your Order page.If you submit your card information and you click the "Place Order" button this page shows up telling you that your order has been placed, and provide you information regarding the TechShop.

# :bulb: Technologies this app shows:

1. State Management.
2. Working with JSON file.
3. Event Handling
